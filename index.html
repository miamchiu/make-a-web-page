1. The basics of Web and HTML

How the Web Works

The web is a bunch of computers that communicate with each other. When a person goes to a web page like www.google.com, their computer sends a HTTP Request to a server. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen. This videodoes a good job of explaining.

HTML

HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". This video gives a good overview.

Tags and Elements

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. This videoexplains the distinction well.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.

Links and Images

To insert a link, <a href="url" or "folder/imagefile > text </a>
To insert a image, <img src="url" alt="text"> alt is to show an alternative text when the link is not available.

HTML Attributes

<TAG ATTR="value">content</TAG>

Boxify design

<div>is to define boxes, within each box, you can assign "class" as a label to the box.
It will look like <div class="Title"></div>

Div and Span

Div: block-level element that takes up entire width of screen.
Span will not push the rest of the element to next line. It will not change the layout of the document.

Id and Class

Ids are used to uniquely identify an element in the entire web page.
Classes are used to style a group of elements similarly.

3. Adding CSS style and HTML structure

To add styles to a document

1. Adding style information into head tag.

<head>
  <link rel="stylesheet" type="text/css" href="style.css">
<head/>

2. Apply style to all of a certain page

p (tag) { }

3. Apply to an element by class

. description { color (attribute) : red (value) }

Box sizing and positioning

Tip 1: Set Box-sizing :(to) border box for every element
Tip2: display:flex; will override this behavior and let divs appear next to each other.

Code, test, refine

1. Look for natural boxes
2. Look for repeated styles and symentic elements
3. Write code; Apply style (from Big to small); Fix things

Avoid Repetition

Give the same style to similar elements, you can reap the benefits of this when you quickly and easily modify all of them at once just by changing a single line of CSS.
